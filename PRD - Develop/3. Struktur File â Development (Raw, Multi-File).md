> \*\*Kenapa `gas-src/` tetap modular padahal target akhirnya 3 file?\*\*\[cite: 2] Google Apps Script sebenarnya otomatis menggabungkan semua file `.gs` dalam satu project ke satu global scope saat runtime вЂ” jadi secara teknis file backend \*\*boleh\*\* tetap terpisah per modul di Apps Script Editor dan tetap jalan normal.\[cite: 2] Tapi karena requirement project ini \*\*strict 3 file total\*\* (`setup.gs`, `code.gs`, `webapp.html`), penulisan kode backend \*\*tetap dilakukan modular per file\*\* (mirip cara frontend dipecah per komponen), lalu \*\*digabung otomatis lewat build script\*\* (`scripts/build-gas.mjs`) вЂ” bukan ditulis manual di satu file besar.\[cite: 2] Ini menjaga kemudahan maintain \& AI-generation per modul, sekaligus tetap memenuhi constraint 3 file saat deploy.\[cite: 2]
