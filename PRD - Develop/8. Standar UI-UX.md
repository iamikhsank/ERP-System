## 8\. Standar UI/UX

* **Aesthetic:** Desain dashboard enterprise modern — *dark sidebar*, *accent color* yang konsisten, *card-based metric widgets*, dan *data table* dengan *sticky header*.
* **Responsive:** Optimasi minimal untuk resolusi desktop (1280px+), namun tetap fungsional dan *usable* di perangkat tablet.
* **Komponen Reusable Wajib:** *Sidebar* navigasi (collapsible), *Topbar* (search, user menu, notifikasi), *Data Table* generik, *Modal form* generik, *Toast notification*, serta *Loading skeleton* saat proses fetch data dari GAS (untuk mengantisipasi latensi bawaan `google.script.run`).
* **Konsistensi Format:** Standar mata uang baku (Rp, dengan *thousand separator*), format tanggal konsisten (DD MMM YYYY), dan penggunaan *status badge* berbasis warna semantik (hijau = approved, kuning = pending, merah = rejected).
